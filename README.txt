► Pour créer le menu et le mettre dans l'ordre:
    - Accéder à l'espace Admin.
    - Aller dans Apparence > Menus.
    - Créer le nom du menu.
    - Ajouter les pages en les cochant puis en cliquant sur le bouton "Ajouter au menu".
    - cocher "Menu Principal" dans Réglages du menu.
    - Enregistrer le menu.

► Pour l'accueil et le blog:
    - Accéder à l'espace Admin.
    - Aller dans Réglages > Lecture.
    - Cocher "Une page statique (choisir ci-dessous)" dans la page d'accueil affiche.
    - selectioner les pages pour la page d'accueil et la page des articles.
    - Enregistrer les modifications.
